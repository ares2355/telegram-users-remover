## Запуск

1. Создайте `.env` с содержимым:

```
CLIENT_API_KEY="..."
CLIENT_API_HASH="..."
CLIENT_API_ID="..."
```
Вместо многоточий вставьте соответсвующие поля [своего клиента](https://my.telegram.org/auth) Telegram.

2. Запустите файл :)
